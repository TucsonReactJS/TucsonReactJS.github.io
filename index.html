Welcome!

The Tucson ReactJS group is a meetup for people who want to deep-dive into React and the Flux application architecture. React beginners and experts are both welcome, and so is anyone who loves and wants to talk about JavaScript.

Transparency rocks, so everything we do will be posted on our Github page
